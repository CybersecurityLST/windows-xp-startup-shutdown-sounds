# 🎵 Windows XP 开关机音效

复刻番茄花园盗版 XP 的经典开关机音效，让你的 Windows 10 / Windows 11 再次拥有那种熟悉的味道。
## 📁本仓库包含两个音频文件：

- poweron.wav —— XP 开机音效
- shutdown.wav —— XP 关机音效

可用于 Windows 10 / Windows 11 的任务计划程序，实现 XP 风格的开关机声音。

## 📜 许可证

音频来自 Windows XP，仅用于学习与怀旧目的。
